# Text to Morse Code Converter

A basic Python script to convert English text into Morse code.

## Description

This command-line script takes a word or phrase (consisting of lowercase English letters) as input from the user and translates it into its corresponding Morse code sequence. It uses a predefined dictionary to map each letter to its standard Morse code representation. The output displays the Morse code with spaces separating the codes for each letter.

## Limitations

*   Currently only supports lowercase English letters (a-z).
*   Does not handle numbers, punctuation, or uppercase letters.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `text to morse.py`.

## Usage

Run the script from your terminal:

```bash
python "text to morse.py"
```

The script will prompt you to "type: ". Enter your word or phrase (in lowercase) and press Enter. The Morse code equivalent will then be displayed.

**Example:**

```
type: hello
.... . .-.. .-.. ---
```
