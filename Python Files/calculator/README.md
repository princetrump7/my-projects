# Simple Python Calculator

A command-line calculator script written in Python.

## Description

This script allows users to perform basic arithmetic operations: addition (`+`), subtraction (`-`), multiplication (`*`), division (`/`), exponentiation (`**`), and modulo (`%`). It prompts the user for two numbers and an operation, then displays the calculated result. The calculator includes error handling for division by zero.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `calculator.py`.

## Usage

Run the script from your terminal:

```bash
python calculator.py
```

The script will prompt you to:
1.  Enter the first number.
2.  Enter the second number.
3.  Enter the desired operation (+, -, *, /, **, %).

It will then display the result or an error message if division by zero is attempted.
