# BMI Calculator

A simple Python script to calculate Body Mass Index (BMI).

## Description

This script prompts the user to enter their height in meters and weight in kilograms. It then calculates the BMI and prints the result, rounded to two decimal places. Based on the calculated BMI, it provides a classification of whether the user is underweight, normal weight, overweight, or obese.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `BMI.py`.

## Usage

Run the script from your terminal:

```bash
python BMI.py
```

The script will then ask for your height and weight, and after you provide the inputs, it will display your BMI and the corresponding classification.
