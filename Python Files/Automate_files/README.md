# Automate Files

A Python script to automatically organize files in a directory based on their extension.

## Description

This script organizes files in the `C:\Users\hp.DESKTOP-Q5PK6DI\OneDrive\Desktop\projects` directory. It moves all files with a `.py` extension into a `Python Files` sub-directory and all files with a `.html` extension into an `HTML Files` sub-directory.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `Automate_files.py`.
3.  Modify the `folder` variable in the script to the directory you want to organize.

## Usage

Run the script from your terminal:

```bash
python Automate_files.py
```

The script will then create the `Python Files` and `HTML Files` directories (if they don't exist) and move the corresponding files.
