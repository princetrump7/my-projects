# Password Generator with GUI

A simple and easy-to-use password generator with a graphical user interface (GUI) built using Python's Tkinter library.

## Description

This script provides a user-friendly way to generate strong, random passwords. The application window contains a single button, "Generate Password". When clicked, it generates a 12-character password composed of a random mix of uppercase letters, lowercase letters, digits, and punctuation symbols. The generated password is then displayed in a text box, from where it can be easily copied.

## Technologies Used

*   Python 3
*   Tkinter (for the GUI)

## Setup

1.  Make sure you have Python installed on your system. Tkinter is usually included with Python, so no extra installation is needed.
2.  Save the script as `password_gui.py`.

## Usage

Run the script from your terminal:

```bash
python password_gui.py
```

A window will appear with a text entry box and a "Generate Password" button. Click the button to generate a new password, which will appear in the text box.
