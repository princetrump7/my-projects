# Simple To-Do List Application (CLI)

A basic command-line interface (CLI) to-do list application written in Python.

## Description

This script provides a straightforward way to manage your daily tasks directly from the terminal. It offers a simple menu-driven interface allowing users to:

1.  **Add a task:** Quickly add new items to your to-do list.
2.  **Remove a task:** Delete completed or unwanted tasks from the list.
3.  **View tasks:** Display all current tasks in your list.
4.  **Quit:** Exit the application.

**Note:** Tasks are stored in memory and are not persistent. They will be lost when the application is closed.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `to_do_app.py`.

## Usage

Run the script from your terminal:

```bash
python to_do_app.py
```

You will be presented with a menu. Enter the number corresponding to the action you wish to perform (1, 2, 3, or 4). Follow the prompts to manage your tasks.
