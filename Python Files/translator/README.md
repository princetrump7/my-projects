# Simple Vowel-to-Exclamation Translator

A basic Python script that performs a unique "translation" by replacing all vowels in a given phrase with exclamation marks.

## Description

This command-line script takes a phrase as input from the user. It then iterates through each letter of the phrase and, if the letter is a vowel (A, E, I, O, U, or their lowercase counterparts), it replaces that vowel with an exclamation mark (`!`). All other characters remain unchanged. The resulting "translated" phrase is then printed to the console.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `translator.py`.

## Usage

Run the script from your terminal:

```bash
python translator.py
```

The script will prompt you to "Enter letter:". Type your desired phrase and press Enter. The "translated" output will be displayed immediately.

**Example:**

```
Enter letter:Hello World
H!ll! W!rld
```
