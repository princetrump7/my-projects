# Random Password Generator

A simple Python script to generate strong, random passwords.

## Description

This script generates a random password of a fixed length (16 characters) using a combination of uppercase letters, lowercase letters, digits (0-9), and common punctuation symbols. The generated password is printed directly to the console.

## Technologies Used

*   Python 3

## Setup

1.  Make sure you have Python installed on your system.
2.  Save the script as `pass generator.py`.

## Usage

Run the script from your terminal:

```bash
python "pass generator.py"
```

A 16-character random password will be displayed in your console. You can easily modify the `password_length` variable within the script to generate passwords of different lengths.
