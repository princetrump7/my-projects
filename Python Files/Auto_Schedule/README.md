# Auto Schedule

A Python script to schedule and display daily coding reminders.

## Description

This script utilizes the `schedule` library to set up daily reminders. It prints a personalized message ("Time to code, Prince!") to the console at specified times (09:00, 14:00, and 20:00). It's a simple way to get gentle nudges to start or continue coding throughout the day.

## Technologies Used

*   Python 3
*   `schedule` library

## Setup

1.  Make sure you have Python installed on your system.
2.  Install the `schedule` library:
    ```bash
    pip install schedule
    ```
3.  Save the script as `Auto_Schedule.py`.

## Usage

Run the script from your terminal:

```bash
python Auto_Schedule.py
```

The script will start and display a message indicating the scheduled reminder times. It will then run continuously, checking for pending tasks every second. You can stop the script by pressing `Ctrl+C`.
