# Project 09: Automation Bot

A Python automation bot for file backups, email notifications, directory monitoring, and scheduled tasks.

## Skills

- schedule
- smtplib
- os / subprocess
- File monitoring
- Config management

## Setup

```bash
pip install -r requirements.txt
python bot.py
```

## Features

- Automated file backups
- Email notifications
- Folder monitoring for changes
- Scheduled task execution
- JSON-based configuration
