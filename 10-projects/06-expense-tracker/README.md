# 06 - Expense Tracker

A CLI expense tracker with CSV persistence, category breakdowns, monthly summaries, and optional chart generation.

## Skills

- Data structures and modeling
- CSV read/write with Python
- Statistics and aggregation
- Date handling and filtering
- Report generation (text and charts)
- matplotlib visualization

## Setup

```bash
pip install -r requirements.txt && python tracker.py
```
