# Project 10: Capstone

A full-stack capstone project demonstrating end-to-end software development skills by solving a real-world problem.

## Skills

- Full-stack development
- Project planning
- Deployment
- Testing
- Documentation

## Setup

```bash
# Backend
cd backend
pip install -r requirements.txt
python app.py

# Frontend
cd frontend
npm install
npm start
```
