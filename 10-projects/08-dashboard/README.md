# Project 08: Dashboard

A responsive data dashboard with interactive charts built using Chart.js, CSS Grid, and mock API data.

## Skills

- Chart.js
- Fetch API
- Responsive design
- CSS Grid
- Real-time updates
- Dark mode

## Setup

```bash
npm install
npm start
```

Open `http://localhost:3000` in your browser.
