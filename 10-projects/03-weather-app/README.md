# Weather App

A browser-based weather application that fetches real-time weather data from the OpenWeatherMap API.

## Skills Covered

- Fetch API
- Async/Await
- JSON parsing
- DOM manipulation

## Setup

```bash
npm install
npm start
```

Then open `http://localhost:3000` in your browser.

## Features Checklist

- [ ] Current weather display (temperature, humidity, wind, description)
