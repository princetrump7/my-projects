# File Organizer

A Python script that automatically organizes files in a directory into categorized folders based on file extensions.

## Skills Covered

- os module
- shutil module
- pathlib
- glob patterns
- Argument parsing

## Setup

```bash
python organizer.py --path /path/to/directory
```
