# 04 - Todo App

A client-side todo application with persistent state management using localStorage.

## Skills

- State management (array-based state, immutable updates)
- localStorage persistence
- CRUD operations
- Event handling (delegation, keyboard events)
- DOM manipulation and rendering

## Setup

```bash
npm install && npm start
```

Then open `http://localhost:3000` in your browser.
